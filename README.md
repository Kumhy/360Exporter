#360Exporter

一个可以把360网盘的内容推送给Aria2下载的小插件

##安装

所有支持userscript的浏览器

- [360Exporter.user.js](https://raw.githubusercontent.com/Kumhy/360Exporter/master/360Exporter.user.js)：下载后拖动360Exporter.user.js至浏览器即可。
